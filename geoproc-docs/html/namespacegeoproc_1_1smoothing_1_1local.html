Bilaplacian operator for mesh smoothing.

Applies the Laplacian smothing operator once with positive value of and then with negative value of for every iteration.

For details on the Laplacian operator see modifier::Laplacian. For details on how it is applied to smoothing see laplacian(const weight&, float, size_t, TriangleMesh&).

Parameters
wThe type of weight used.
lambdaSmoothing parameter.
n_iterNumber of iterations of the algorithm.
[out]mMesh to be smoothed.
Precondition
The mesh requires:
  • Neighbourhood data (see TriangleMesh::make_neighbourhood_data)
  • Angles and areas (see TriangleMesh::make_angles_area)
The mesh can't have boundaries if the weight::cotangent is applied. However, this is not checked since the mesh need not have its boundaries computed.
