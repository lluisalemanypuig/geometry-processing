Detailed Description

Local smoothing algorithms core functions.

Here are defined the core functions that the algorithms in namespace local (and others too) use.

For vertex vi computes where is the valence of vertex vi (the number of neighbours in its one-ring). This value is value is assigned to all its neighbours in the mesh in the output parameter pv_ws.
