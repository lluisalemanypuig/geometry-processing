Remeshes a triangle mesh using harmonic map coordinates.

First, computes the harmonic maps parametrisation of the mesh, which yields a set of 2d coordinates in the xy plane, more concretely in the square [0,1]x[0,1] (regardless of the shape).

Using the 2d coordinates calculated calls harmonic_maps(const TriangleMesh&,size_t,size_t,const std::vector<glm::vec2d>&,TriangleMesh&);

Parameters
[in]mTriangle mesh to be remeshed.
[in]NSize of the grid in the x-axis.
[in]MSize of the grid in the y-axis.
[in]wType of weight to use to obtain the parametrisation.
[in]sBoundary shape of the parametrisation.
[out]rmRemeshed mesh.
Precondition
The input mesh needs all of its edges computed, namely, neighbourhood data has to be valid.
The preconditions of the method parametrisation::harmonic_maps have to be true before calling this method.
Only boundary_shape::Square is allowed.
Returns
Returns false on failure. Returns true on success.
