Detailed Description

Implementation of different curvature algorithms.

The following curvatures are available:

  • curvature::Gauss(const TriangleMesh&, std::vector<double>&, double*, double*).
  • curvature::Gauss(const TriangleMesh&, std::vector<double>&, size_t).
  • curvature::Gauss(const TriangleMesh&, std::vector<double>&, size_t, double*, double*).
  • curvature::mean(const TriangleMesh&, std::vector<double>&, double*, double*).
  • curvature::mean(const TriangleMesh&, std::vector<double>&, size_t).
  • curvature::mean(const TriangleMesh&, std::vector<double>&, size_t, double*, double*).

void geoproc::curvature::Gauss ( const TriangleMesh &  mesh,
std::vector< double > &  Kg,
double *  min = nullptr,
double *  max = nullptr 
)

Computes the Gauss curvature for each vertex.

Use the discretisation of the Laplace-Beltrami operator, using the cotangents as weights.

This function needs the meshe's angles and areas (see TriangleMesh::make_angles_area).

Parameters
meshInput mesh.
[out]KgAn approximation of the Gauss curvature per vertex.
[out]minMinimum curvature value.
[out]maxMaximum curvature value.
Precondition
The mesh requires:
  • Neighbourhood data (see TriangleMesh::make_neighbourhood_data)
  • Angles and areas (see TriangleMesh::make_angles_area)
