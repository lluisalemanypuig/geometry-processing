Detailed Description

Vertex iterator class.

Iterates over the vertices on the 1-ring neighbourhood of a given vertex.

Initialise iterator at a vertex.

Modifies this class so that current returns the first vertex on the 1-ring neighbourhood of v.

Parameters
vIndex of the vertex. Must be a valid vertex index: 0 <= v < number of vertices.
Returns
Returns the first iterated vertex index. If this value is not collected, call current, before calling next.

◆ next()

int geoproc::iterators::vertex::vertex_vertex_iterator::next ( )
virtual

Returns the next vertex.

The vertex index returned is in the 1-ring neighbourhood of the vertex used to initialise the iterator (see init).

The order of the vertices is guaranteed to be counterclockwise.

Precondition
This method must be called after initialising the iteratior with init.
Returns
Returns a valid vertex index.

Implements geoproc::iterators::mesh_iterator.
