Detailed Description

Main namespace of the library.

In this namespace is defined the main data structure used to implement all the algorithms: the TriangleMesh.

In order to read a mesh from disk, see namespace PLY_reader.

All the algorithms are defined in their corresponding namespaces.

◆ boundary_shape

enum geoproc::boundary_shape : int8_t
strong

Shapes for boundary vertices.

Sometimes we may want to place the vertices at the boundary of a mesh in a certain shape on the xy plane. The shapes are the following:

  • circle: see boundary_shape::Circle.
  • square: see boundary_shape::Square.
Enumerator
Circle 

The boundary vertices are placed on the circumference of a circle of radius 1, centered at (0,0).

Square 

The boundary vertices are placed on the boundary of a square of side length 2, centered at (0,0).
