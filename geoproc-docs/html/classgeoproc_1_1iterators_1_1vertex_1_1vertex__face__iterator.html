Detailed Description

Face iterator class.

Iterates over the faces on the 1-ring neighbourhood of a given vertex. These are the faces adjacent to the given vertex.

◆ next()

int geoproc::iterators::vertex::vertex_face_iterator::next ( )
virtual

Returns the next face.

The face index returned is in the 1-ring neighbourhood of the vertex used to initialise the iterator (see init).

The order of the faces is guaranteed to be counterclockwise.

Precondition
This method must be called after initialising the iteratior with init.
Returns
Returns a valid face index.
